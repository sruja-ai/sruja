## Data Lifecycle Stages

### Hot Data
- Frequently accessed
- High-performance storage
- Low latency
- High cost

### Warm Data
- Occasionally accessed
- Standard storage
- Moderate latency
- Moderate cost

### Cold Data
- Rarely accessed
- Archive storage
- High latency
- Low cost

### Archive Data
- Long-term storage
- Deep archive
- Very high latency
- Very low cost
