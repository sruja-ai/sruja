## Purpose

These tools are for **local development productivity only**, not for publishing:

- ✅ Syntax highlighting
- ✅ Code snippets
- ✅ Basic file association
- ✅ Integration with Go CLI tool

**Not included** (for published extension):
- ❌ Full LSP server
- ❌ Advanced diagnostics
- ❌ Code completion

Use the Go CLI tool (`sruja compile`, `sruja validate`) for advanced features.
