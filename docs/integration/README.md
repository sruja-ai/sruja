## Overview

Sruja integrates with various external systems and protocols to enable:

- **AI Integration** - MCP protocol for LLM assistants (Cursor, Copilot, etc.)
- **Notebook Integration** - Architecture notebooks with MCP tools (see [Notebooks](../notebooks/README.md))
- **CI/CD Integration** - Git hooks and validation pipelines
- **Editor Integration** - LSP for IDE support
- **Version Control** - Git as source of truth
