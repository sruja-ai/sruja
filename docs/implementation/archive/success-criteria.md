# Success Criteria

## Phase 1: Core Data Transformations (Go) ✅

* [ ] DSL → JSON works for all element types
* [ ] JSON → DSL works for all element types
* [ ] Round-trip tests pass (DSL → JSON → DSL)
* [ ] Imports preserved in round-trip
* [ ] CLI commands work
* [ ] 90%+ test coverage

## Phase 2: HTML Export (Go) ✅

* [ ] HTML export generates valid files
* [ ] HTML opens in browser
* [ ] Links to JSON correctly
* [ ] Supports CDN mode

## Phase 3: JavaScript Library (TypeScript) ✅

* [ ] Loads and renders JSON
* [ ] All view types work
* [ ] Interactions work (click, hover, zoom, pan)
* [ ] Documentation panel works
* [ ] Search works
* [ ] Can export as PNG from standalone HTML
* [ ] Can export as SVG from standalone HTML
* [ ] Export works in all modern browsers

## Phase 4: Web Studio (TypeScript) ✅

* [ ] Can load JSON
* [ ] Can add/edit/delete elements
* [ ] Can create relations
* [ ] Can export to DSL (via WASM)
* [ ] Can export to JSON
* [ ] Can export to HTML
* [ ] Can export as PNG from Studio
* [ ] Can export as SVG from Studio
* [ ] Export options (full diagram, high-resolution, background color)
* [ ] Works offline (no backend)
