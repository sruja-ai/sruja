# System Design 101: Fundamentals

## Overview
- Establish core system design principles and a shared vocabulary
- Learn how to clarify requirements and model systems effectively

## Learning Goals
- Identify functional vs non-functional requirements
- Use building blocks: services, datastores, queues, caches, APIs
- Model systems and flows for clarity and testability
- Prepare for production readiness: logging, metrics, resiliency, deployment

## Prerequisites
- Basic knowledge of web services and databases

## Course Structure
- Module 1: Fundamentals
- Module 2: Building Blocks
- Module 3: Advanced Modeling
- Module 4: Production Readiness

## Where to Start
- Begin with Module 1 and proceed in order; each module builds on the previous
