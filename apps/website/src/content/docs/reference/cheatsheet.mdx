# Sruja Cheatsheet

## Elements

```sruja
architecture "Shop" {
  person User
  system App {
    container Web
    container API
    datastore DB
  }
  User -> App.Web "Uses"
  App.Web -> App.API "Calls"
  App.API -> App.DB "Reads/Writes"
}
```

## Component

```sruja
system App {
  container Web {
    component Cart
  }
}
```

## Scenario

```sruja
scenario Checkout {
  User -> App.Web "adds items"
  App.Web -> App.API "validates"
  App.API -> App.DB "stores order"
}
```

## Deployment

```sruja
deployment Prod {
  node Cloud {
    node Region {
      node Service {
        containerInstance Web
      }
    }
  }
}
```
